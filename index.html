Intended Audience

Application developers who wish to provide consistent scrollable interface experiences across all Web-based platforms.

What is ScrollView?

ScrollView provides a cross-browser scrollable content widget.


Supported Browsers

Everything! Modern browsers, as well as legacy (IE6+), Android (2.3+), and mobile Safari.

Y.Widget

What is it?

Y.Widget is a foundation class that many UI components in YUI are built on top of.

What does it provide?

  • Events, Attributes, & Plugins
  • Render Lifecycle Methods
  • Abstract Rendering Methods

Extending Widget

/* MyWidget class constructor */
function MyWidget(config) {
    MyWidget.superclass.constructor.apply(this, arguments);
}

/* Identify the Widget class */
MyWidget.NAME = "MyWidget";

    // Attributes defined w/ default values and get/set functions

    // Methods/properties to add to the prototype of the new class
});

Instantiating ScrollView


YUI().use('scrollview', function (Y) {

	var scrollView = new Y.ScrollView({
		srcNode: '#someContainer',
		height: 400
	});

	scrollView.render();
});

Global State

Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the background.

Additionally custom events can be triggered on a per slide basis by binding to the data-state name.
